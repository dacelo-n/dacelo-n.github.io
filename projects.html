Password Permutations

I created a password permutator to allow a user to enter in key words, and have a list produced of all the casing permutations for the said words. For example : 'dog' will produce ['dog','Dog','dOg','DOg','doG','DoG','dOG','DOG']. 8 possibilities exist without rearranging the order of the letters, thus maintaining the original word, because there are 3 letters and each letter has only 2 options : lower or upper case. This can also put two or more words together in different permutations for example : 'dog' and 'cat' can be 'dogcat' or 'catdog' with every possibility as well. Numbers are not affected. dogcat100, 100catdog, 100dogcat, cat100dog, dog100cat , etc etc. This can be used to generator passwords when pentesting or for educational password cracking purposes.
